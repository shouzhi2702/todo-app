# 任务分类与统计仪表盘方案

> 作者：shouzhi | 创建时间：2026-03-28 | 分支：feature/category-dashboard

## 一、需求描述

在现有 Todo App 上新增两个功能：

### 1. 任务分类
- 新增分类维度：工作、学习、生活、其他
- 创建待办时可选择分类
- 首页支持按分类筛选待办事项

### 2. 统计仪表盘
- 新增 `/dashboard` 页面
- 展示：总任务数、已完成数、完成率
- 按分类展示任务分布
- 简洁直观的数据展示

## 二、技术方案

### 数据层
- 新增 `Category` 枚举（WORK, STUDY, LIFE, OTHER）
- `Todo` 实体增加 `category` 字段，默认 OTHER
- `TodoRepository` 增加按分类查询和统计方法

### 服务层
- `TodoService` 增加按分类查询、统计方法
- 统计数据：总数、已完成数、各分类计数

### 控制层
- `TodoController` 首页增加分类筛选参数
- 新增 `DashboardController` 处理 `/dashboard` 路由

### 视图层
- `index.html` 新增分类选择下拉框 + 分类筛选按钮
- 新增 `dashboard.html` 统计仪表盘页面
- 导航栏增加"仪表盘"链接
